Definition for Version Control.

Version control, also known as source control, is the practice of tracking and managing changes to software code.

Version control systems are software tools that help software teams manage changes to source code over time.

Git

Git is a version control system that was created for Linux develpment by Linus Torvalds.

Git is a distributed system that keeps a repository with complete version history in every directory.

Git's functions are activated by text commands such as git add and git commit.

How to commit a project to Github repository

Open terminal

git init

git add

git commmit -m "first commit"
